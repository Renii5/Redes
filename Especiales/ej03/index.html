Este es el div contenedor

Ayudas para el ejercicio

El div contenedor debe desactivarse cuando se active la ventana modal.

Para desactivarlo, se le podría aplicar transparencia y deshabilitar todos sus botones.

Desactivado:

  • pointer-events: none
  • opacity: 0.3

Activado:

  • pointer-events: auto
  • opacity: 1
